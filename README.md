# AI Chatbot Project

## Description
This is a simple AI chatbot developed using Python. It interacts with users by accepting text input and generating appropriate responses.

## Technologies Used
- Python

## Features
- Interactive command-line chatbot
- User-friendly interface
- Easy to customize and extend

## How to Run
1. Install Python 3.x.
2. Open a terminal in the project folder.
3. Run:
   python chatbot.py

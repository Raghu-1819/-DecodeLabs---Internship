# Rule-Based AI Chatbot

A simple Rule-Based AI Chatbot developed using Python. This chatbot responds to predefined user inputs using conditional statements (`if-elif-else`) and runs continuously until the user exits the conversation.

---

## Features

* Greets the user with predefined responses
* Answers simple questions
* Handles exit commands (`bye`, `exit`, `quit`)
* Uses `if-elif-else` decision-making logic
* Runs in a continuous loop
* Beginner-friendly Python project

## Installation

1. Clone this repository:

```bash
git clone https://github.com/your-username/Rule-Based-AI-Chatbot.git
```

2. Navigate to the project folder:

```bash
cd Rule-Based-AI-Chatbot
```

3. Run the chatbot:

```bash
python chatbot.py
```
